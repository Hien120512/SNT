Software Engineer – User Interface

You will be part of the Application Framework team responsible for designing and developing flexible and extensible application layer framework as well as functional components.  You will use best-of-breed technology from both established vendors such as Microsoft as well as open source technology.  As a Software Engineer, this will be an opportunity to play a key role in designing, developing, and furthermore innovating together as you bring your considerable software engineering skills to our dynamically growing organization.
 
Additionally, as a Software Engineer, you will focus on the following areas:
  • Collaborate with engineers, Product/Project Managers and Quality Assurance engineers to develop multi-tier, multi-layered application layer framework and functional components, following coding, documentation and design standards
  • Innovate, design and develop extensible and reusable applications to specification, for the enterprise suite
  • Participate in internal/cross team meetings, requirements gathering, scoping, decision making and technical documentation
The Software Development group focuses on developing and building cutting edge, high performance enterprise software solutions that deliver risk quantification tools to our 400+ clients worldwide.  Using cutting edge technology, we are building an exciting next generation product that leverages the latest advances in database tools, grid computing, hardware-based acceleration techniques, and visualization optimization. Our Enterprise Platform utilizes restful web services, 100+ TB of data warehousing/OLAP solutions, large scale distributed data processing technology, best-in-class virtualization technology, and GPU-based computing in a SaaS environment.
 
To be successful in the role of Software Engineer, you must be/have:
  • BS/MS degree in Computer Science or related field with 6+ years of professional application development experience in Windows environment
  • 2+ years professional experience developing multi-tier, scalable, and high performance applications using C# and other .NET technologies, WPF, Windows Forms, Smart Parts, ASP.NET
  • Comprehensive knowledge of design patterns. Ability to pick up new technologies to suit the application needs.
  • Experience working with .NET application framework such as Composite Application library, Unity, Spring.NET, or building home-brewed application framework
  • Experience designing and developing advanced .NET custom UI components
  • Solid understanding of SOA and database
  • Demonstrated ability to produce quality code
  • Work well with others in fast-paced, iterative product definition and development environment
Additionally, RMS will give additional consideration to candidates that have skills or knowledge in the following areas:
  • Experience with SDK level development with business intelligence platforms like MicroStrategy, Business Objects, Hyperion
  • Extra points if you had developed Silverlight or Flex components
