Objective of the Manager for QA:  To lead the QA team of around 25 people in the RMS India office working on manual and automated QA of an enterprise scale application.
 
Key Accountabilities & Deliverables:
 
  • Interface with developers to understand new features and enhancements.
  • Create test plans, test schedules and test cases for assessing these changes or enhancements.
  • Set up test environments.
  • Execute tests, report bugs and other issues found.
  • Manage test personnel, assets and test metrics.
  • Automate tests as appropriate.  Create and maintain tools to improve productivity and reduce error.
  • Be an effective mentor and coach.
  • Gain and utilize an in-depth knowledge of products and relevant technologies
  • Goal definition, Goal assessment, Performance management, Skill management, resource management
  • Coordinate with support teams like IT, HR, Admin on day to day level
  • Support in hiring process with interview, short listing, and coordination.
  • Clearly communicate project status and directions
