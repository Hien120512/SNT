Senior Software Engineer - WPF / Silverlight

San Francisco Area (Newark, CA, United States) Position: 11035
Geospatial Software Development group provides location intelligence to RMS software products. The team is responsible for design, implementation, and maintenance of software solutions that process geospatial data and produce visual and non-visual information critical for decision analytics in RMS software products.  This includes geocoding, site-specific hazard data retrieval, and map visualization as key software sub-systems developed by the team.

Additionally, as a­­­­­­­­­­­­ Senior Software Engineer, you will focus on the following areas:
  • Design and implement server-side, robust high volume map generation software using ESRI ArcGIS and latest Microsoft development technologies
  • Develop prototypes quickly for new requirements
  • Optimally integrate web mapping APIs from Google and Bing with RMS maps
  • Apply specialized knowledge of spatial data formats and storage to develop advanced mapping capabilities
The Software Development group focuses on developing and building cutting edge, high performance enterprise software solutions that deliver risk quantification tools to our 400+ clients worldwide.  Using cutting edge technology, we are building an exciting next generation product that leverages the latest advances in database tools, grid computing, hardware-based acceleration techniques, and visualization optimization. Our Enterprise Platform utilizes restful web services, 100+ TB of data warehousing/OLAP solutions, large scale distributed data processing technology, best-in-class virtualization technology, and GPU-based computing in a SaaS environment.
 
Skills and Experience Required:

To be successful in the role of Senior Software Engineer, you must be/have:
  • BS or MS in Computer Science or related technical fields or equivalent
  • Strong programming with, C#, Silverlight/WPF and Microsoft .NET framework
  • Extensive usage of web mapping APIs like Bing Maps, Google Maps, and ESRI Maps, etc.
  • Familiar with UI design methodologies and frameworks
  • Develop and use multi-user web based mapping services
  • Mapping related software development experience with passion for developing cutting edge geospatial software
  • Understanding of cartographic concepts
Preferred but not Required Skills: 
  • Understanding of foundation of mainstream mapping frameworks like ArcGIS is a plus
  • Understand different spatial data formats like Shapefiles, KML, SQL Server Spatial etc is a plus
